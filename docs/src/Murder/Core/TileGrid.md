#### Resize(IntRectangle)
```csharp
public void Resize(IntRectangle rectangle)
```

This supports resize the grid up to:
              _____      ______
             |     | -&gt; |      |
             |_____x    |      |
                        |______x
            or
              _____      _____
             |  x  | -&gt; |  x  |
             |_____|    |_____|
            
            Where x is the bullet point.
